About This Project

This website is a fan-made tribute to the universe of Avatar: The Last Airbender and The Legend of Korra. It uses the public Last Airbender API to showcase beloved characters, bending arts, and elemental powers. This website also is a short, little, fun project to see which element you are destined to be.

On this website, You'll find not only a form/quiz that was assigned but a whole interactive character gallery, detailed information about the bending elements and their sub-styles, and immersive visuals inspired by the shows.


A fun fact actually. Our original project was a harry potter themed form/quiz, but we thought of something else that was also divided into four housing/lifes. We chose Avatar since it could be sorted like the four houses in harry potter. (Also only one of the group members (Nicholas) has seen all the harry potter movies).
